Selecting a Printer

If you have multiple printers that you are controlling with MatterControl it is very easy to switch between them.

  1. Navigate to the 'Start' Page
  2. From the 'Printers...' list select the printer you want to use
  3. You can now select an 'Empty Bed' (or an existing plate) to connect to your printer and start adding things to plate
